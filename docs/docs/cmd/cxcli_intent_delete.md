# cxcli intent delete

Deletes an intent in an agent

```
cxcli intent delete [intent-name] [flags]
```

## Options

```

  -h, --help                 help for delete
  -l, --location-id string   Dialogflow CX Location ID of the Project
  -p, --project-id string    Dialogflow CX Project ID
```
